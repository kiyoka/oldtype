

     A tDiary to OldType data convertion tool


 This tool supports tDiary's entry and comments conversion.

1. How to use
 1) Put cache data of tDiary to ./tdiary_data

 2) Execute ./tdiary2oldtype.sh
    $ ./tdiary2oldtype.sh

2. Generated contens.
  This tool generate some contents as OldType format.
  like xxxx.2008_10_10.ot files.
  ( xxxx is your ${USER} environment variable. )

  Please commit to subversion repository of you OldType Wiki site.
